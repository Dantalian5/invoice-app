## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

inVoice is a full-stack web application designed to simplify invoice management. It provides features such as creating, reading, updating, and deleting invoices, along with form validation when adding or editing an invoice. Users can save draft invoices, mark pending ones as paid, and filter invoices by status (draft, pending, or paid). The app also includes light and dark mode toggling for user preference, responsive design to ensure an optimal layout across all screen sizes, and hover states for all interactive elements to enhance usability.

The project leverages a reactive and modern stack: the frontend is implemented with React to ensure a dynamic and seamless user experience, while the backend is built using Node.js, offering a robust RESTful API for handling data operations. Together, these technologies provide a reliable and efficient solution for managing invoices in a professional and user-friendly way.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Dantalian5/invoice-app.git
   ```

